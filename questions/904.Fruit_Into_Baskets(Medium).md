## 904. Fruit Into Baskets (Medium)
**Date and Time:** Jun 21, 2026

Link: https://leetcode.com/problems/fruit-into-baskets/

<br>

### Question:
You are visiting a farm that has a single row of fruit trees arranged from left to right. The trees are represented by an integer array `fruits` where `fruits[i]` is the **type** of fruit the `i`th tree produces.

You want to collect as much fruit as possible. However, the owner has some strict rules:

* You only have **two** baskets, and each basket can only hold a **single type** of fruit. There is no limit on the amount of fruit each basket can hold.
* Starting from any tree of your choice, you must pick **exactly one fruit** from every tree (including the start tree) while moving to the right. The picked fruits must fit in one of your baskets.
* Once you reach a tree with fruit that **cannot** fit in your baskets, you must stop.

Given the integer array `fruits`, return the **maximum** number of fruits you can pick.

<br>

**Example 1:**
> **Input:** fruits = [1,2,1]
> 
> **Output:** 3
>
> **Explanation:** We can pick from all 3 trees.

**Example 2:**
> **Input:** fruits = [0,1,2,2]
> 
> **Output:** 3
>
> **Explanation:** We can pick from trees [1,2,2]. If we had started at the first tree, we would only pick from trees [0,1].

**Example 3:**
> **Input:** fruits = [1,2,3,2,2]
> 
> **Output:** 4
>
> **Explanation:** We can pick from trees [2,3,2,2]. If we had started at the first tree, we would only pick from trees [1,2].

<br>

#### Constraints:
* `1 <= fruits.length <= 10^5`

* `0 <= fruits[i] < fruits.length`

<br>

### Walk-through:
Sliding window with `basket {fruit_type: count}` that holds at most 2 types.

1. Expand `r` right. If `fruits[r]` is not already in `basket` and `basket` is full (`len == 2`), shrink from the left: decrement `basket[fruits[l]]`, delete the entry if its count hits 0, advance `l`.
2. Add `fruits[r]` to `basket` and update `res = max(res, r - l + 1)`.

<br>

### My Solution:
```python
class Solution:
    def totalFruit(self, fruits: List[int]) -> int:
        # Q: Find max # of trees we can pick given we have only 2 baskets
        # S: Sliding window with set() to save 2 types, update res = r-l+1 everytime except we need to update l
        # TC: O(n), n=len(fruits), SC: O(1)
        l = 0
        res = 0
        basket = {}     # {basket_i: cnt}
        for r in range(len(fruits)):
            # Check if we can pick this fruit
            while fruits[r] not in basket and len(basket) == 2:
                basket[fruits[l]] -= 1
                if basket[fruits[l]] == 0:
                    del basket[fruits[l]]
                l += 1
            # Otherwise, directly update res   
            basket[fruits[r]] = basket.get(fruits[r], 0) + 1
            res = max(res, r-l+1)
        return res
```
**Time Complexity:** $O(n)$, where `n = len(fruits)` <br>
**Space Complexity:** $O(1)$, `basket` holds at most 2 entries
