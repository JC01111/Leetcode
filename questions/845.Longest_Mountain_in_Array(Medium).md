## 845. Longest Mountain in Array (Medium)
**Date and Time:** Jun 4, 2026

Link: https://leetcode.com/problems/longest-mountain-in-array/

<br>

### Question:
You may recall that an array `arr` is a **mountain array** if and only if:

* `arr.length >= 3`
* There exists some index `i` (0-indexed) with `0 < i < arr.length - 1` such that:
  * `arr[0] < arr[1] < ... < arr[i - 1] < arr[i]`
  * `arr[i] > arr[i + 1] > ... > arr[arr.length - 1]`

Given an integer array `arr`, return _the length of the longest subarray, which is a mountain_. Return `0` if there is no mountain subarray.

<br>

**Example 1:**
> **Input:** arr = [2,1,4,7,3,2,5] <br>
> **Output:** 5 <br>
> **Explanation:** The largest mountain is [1,4,7,3,2] which has length 5.

**Example 2:**
> **Input:** arr = [2,2,2] <br>
> **Output:** 0 <br>
> **Explanation:** There is no mountain.

<br>

#### Constraints:
* `1 <= arr.length <= 10^4`

* `0 <= arr[i] <= 10^4`

<br>

### Walk-through: 
Use a **single pass** with a pointer `i` acting as the base of each potential mountain.

1. From each `base`, climb while `arr[i] < arr[i+1]`. If `base == i` after climbing, no ascent was found — advance and continue.

2. Record the `peak`. Descend while `arr[i] > arr[i+1]`. If `peak == i` after descending, no descent was found (plateau or end) — advance and continue.

3. Otherwise, a valid mountain `[base..i]` was found. Update `res = max(res, i - base + 1)`.

<br>

### Two-pass DP:
```python
class Solution:
    def longestMountain(self, arr: List[int]) -> int:
        # Use two dps to maintain the subarray from its left dp[:l] and right dp[r+1:]
        # For left dp, check if arr[i] < arr[i+1], if so, update dp[i] = dp[i-1]+1, leftDp[i]: maxsubarray Sum before i
        # For right dp, if arr[i] > arr[i+1], if so, update dp[i] = dp[i+1]+1, rightDp[i]: maxsubarray Sum after i
        # When filling right dp, maintain a var to keep track of the current maxSubarray by left_dp[i] + right_dp[i] + 1
        # TC: O(n), n=len(arr), SC: O(n)
        leftDp, rightDp = [0] * len(arr), [0] * len(arr)
        res = 0
        # Left pass
        for i in range(1, len(arr)):
            if arr[i] > arr[i-1]:
                leftDp[i] = leftDp[i-1] + 1
        # Right pass
        for i in range(len(arr)-2, 0, -1):
            if arr[i] > arr[i+1]:
                rightDp[i] = rightDp[i+1] + 1
            # Update res only if we have both left and right subarray > 0
            if leftDp[i] > 0 and rightDp[i] > 0:
                res = max(res, rightDp[i] + leftDp[i] + 1)
        return res

        # leftDp = [0, 0, 0]
        # rightDp = [0, 0, 0]
        # res = 0
```
**Time Complexity:** $O(n)$, `n = len(arr)`, each element is visited at most twice. <br>
**Space Complexity:** $O(n)$

<br>

### Climbing simulation:
```python
class Solution:
    def longestMountain(self, arr: List[int]) -> int:
        # Q: Return the length of longest mountain
        # S: 1. Find base and peak
        # Loop over arr for every index i as a base, first start climbing, check every arr[i] < arr[i+1], when the first while loop breaks, i is either peak or i is not a valid peak
        # 2. If valid peak, we start counting descending by arr[i] > arr[i+1], until the first while loop breaks, i is either the same peak or new base
        # 3. Update res
        # TC: O(n), n=len(arr), SC: O(1)
        res = 0
        i = 0
        while i < len(arr):
            # Start with a base
            base = i
            # Check if we are climbing
            while i+1 < len(arr) and arr[i] < arr[i+1]:
                i += 1
            # Check if arr[i] == arr[i+1] or arr[i] > arr[i+1], so we need to udpate base to the next index
            if base == i:
                i += 1
                continue
            # Otherwise, we have found a peak
            peak = i
            # Check if we are descending
            while i+1 < len(arr) and arr[i] > arr[i+1]:
                i += 1
            # Check if we have passed peak
            if peak == i:
                i += 1
                continue
            # Update res
            res = max(res, i - base + 1)
        return res
```
**Time Complexity:** $O(n)$, `n = len(arr)`, each element is visited at most twice. <br>
**Space Complexity:** $O(1)$, only constant extra space used.
