## 253. Meeting Rooms II (Medium)
**Date and Time:** Nov 21, 2024, 16:25 (EST)

Link: https://leetcode.com/problems/meeting-rooms-ii

<br>

### Question:
Given an array of meeting time `intervals` intervals where `intervals[i] = [start_i, end_i]`, return the minimum number of conference rooms required.

<br>

**Example 1:**
> **Input:** intervals = [[0,30],[5,10],[15,20]]
> 
> **Output:** 2

**Example 2:**
> **Input:** intervals = [[7,10],[2,4]]
> 
> **Output:** 1

**Edge Case:**
> **Input:** intervals = [[1,5],[8,9],[8,9]]
> 
> **Output:** 2

<br>

#### Constraints:
* `1 <= intervals.length <= 10^4`

* `0 <= start_i < end_i <= 10^6`

<br>

### Walk-through: 
1. First, we sort `intervals`.

2. We use `minHeap[]` to save every room's end time. Then, for every interval, we compare their end time with the current minimum end time in `minHeap[0]`. If `start >= minHeap[0]`, we `pop()` the smallest end time room from `minHeap[]`, and append the new end time to `minHeap`, otherwise, we should just append the new end time to `minHeap`.

<br>

### Python Solution:
```python
class Solution:
    def minMeetingRooms(self, intervals: List[List[int]]) -> int:
        # Q: Return the min rooms we need
        # Save the end time for each meeting
        # S: Compare new start_j with previous end_i, if start_j < end_i, we need a new room
        # 1. Sort intervals by start
        # 2. Maintain a minHeap for end time from intervals
        # 3. Compare new start_j with previous end_i, if start_j < end_i, we need a new room, rooms += 1
        # TC: O(nlogn), n=len(intervals), SC: O(n)
        intervals.sort()
        minHeap = []
        # 2. Save end into minHeap
        for start, end in intervals:
            heapq.heappush(minHeap, end)
        # 3. Compare new start_j with the top of minHeap end time
        rooms = 1
        for i in range(1, len(intervals)):
            start, end = intervals[i]
            # If new start time < min end time from minHeap
            if start < minHeap[0]:
                rooms += 1
            else:
                heapq.heappop(minHeap)
        return rooms
```
**Time Complexity:** $O(nlogn)$ <br>
**Space Complexity:** $O(n)$

### Python Solution:
May 28, 2026, [Time Taken 25m]
```python
class Solution:
    def minMeetingRooms(self, intervals: List[List[int]]) -> int:
        # Maintain a minHeap for all the meeting end time
        # Everytime we compare an interval [start, end]
        # If start >= minHeap[0], means we can keep releasing room
        # keep room 30 and keep room 10, then release 10, and keep room 20
        # TC: O(nlogn), SC: O(n)
        minHeap = []
        intervals.sort()
        # Start comparing and updating maxRooms
        maxRooms = 0
        for start, end in intervals:
            # Removing all the expired rooms
            while minHeap and start >= minHeap[0]:
                heapq.heappop(minHeap)
            heapq.heappush(minHeap, end)
            maxRooms = max(maxRooms, len(minHeap))
        return maxRooms
```
**Time Complexity:** $O(nlogn)$ <br>
**Space Complexity:** $O(n)$
