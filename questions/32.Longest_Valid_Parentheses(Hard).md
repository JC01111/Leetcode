## 32. Longest Valid Parentheses (Hard)
**Date and Time:** Jun 3, 2026

Link: https://leetcode.com/problems/longest-valid-parentheses/

<br>

### Question:
Given a string containing just the characters `'('` and `')'`, return _the length of the longest valid (well-formed) parentheses substring_.

<br>

**Example 1:**
> **Input:** s = "(()" <br>
> **Output:** 2 <br>
> **Explanation:** The longest valid parentheses substring is "()".

**Example 2:**
> **Input:** s = ")()())" <br>
> **Output:** 4 <br>
> **Explanation:** The longest valid parentheses substring is "()()".

**Example 3:**
> **Input:** s = "" <br>
> **Output:** 0

<br>

#### Constraints:
* `0 <= s.length <= 3 * 10^4`

* `s[i]` is `'('`, or `')'`.

<br>

### Walk-through: 
Use a **stack** initialized with `[-1]` as a sentinel index.

1. For each character, if it is `'('`, push its index onto the stack.

2. If it is `')'`, pop the top of the stack.
   - If the stack is **not empty**, the current valid length is `idx - stack[-1]`, update `res`.
   - If the stack is **empty**, push the current index as the new sentinel (boundary for the next valid substring).

<br>

### Python Solution:
```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        # Q: Return the longest valid parentheses substring
        # S: 1. Save "(" index into stack[]
        # 2. If i == ")", we pop stack[], if stack is not empty, calculate the
        # difference between current index and stack[-1]. Else, save current ")"
        # index into stack[]
        # TC: O(n), n=len(s), SC: O(n)
        stack = [-1]
        res = 0
        for idx, n in enumerate(s):
            if n == "(":
                stack.append(idx)
            else:
                # Pop the last valid
                stack.pop()
                if stack:
                    # Pop the last "("
                    res = max(res, idx - stack[-1])
                # Index the current ")" as new boundary
                else:
                    stack.append(idx)
        return res
```
**Time Complexity:** $O(n)$, `n = len(s)`, single pass through the string. <br>
**Space Complexity:** $O(n)$, for the stack.
