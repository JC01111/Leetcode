## 437. Path Sum III (Medium)
**Date and Time:** Nov 11, 2024, 21:52 (EST)

Link: https://leetcode.com/problems/path-sum-iii/

<br>

### Question:
Given the `root` of a binary tree and an integer `targetSum`, return the number of paths where the sum of the values along the path equals `targetSum`.

The path does not need to start or end at the root or a leaf, but it must go downwards (i.e., traveling only from parent nodes to child nodes).

<br>

**Example 1:**

<img src="../images/437.jpg" width=400>

> **Input:** root = [10,5,-3,3,2,null,11,3,-2,null,1], targetSum = 8
> 
> **Output:** 3
>
> **Explanation:** The paths that sum to 8 are shown.

**Example 2:**
> **Input:** root = [5,4,8,11,null,13,4,7,2,null,null,5,1], targetSum = 22
> 
> **Output:** 3

<br>

#### Constraints:
* The number of nodes in the tree is in the range `[0, 1000]`.

* `-10^9 <= Node.val <= 10^9`

* `-1000 <= targetSum <= 1000`

<br>

### Walk-through: 
**Non-optimized Solution:**

Similar to Path Sum II and Path Sum, we run the algorithm on each node to find path sum until `curSum == targetSum`, we can increment `res += 1`.

<br>

**Prefix Sum Solution:**

Similar to [560.Subarray Sum Equals K](./560.Subarray_Sum_Equals_K(Medium).md), we build a `hashmap{}` to keep track of each `curSum` with their counts. When `diff = curSum - targetSum` exists in `hashmap{}`, that means we have ways to get the `targetSum`, so we update `res += hashmap[diff]`.

Decrement `hashmap[curSum]` by `1` after we finished traversing node's left, right subtrees. Because we don't want other subtree to use the count of this prefixSum in our current subtree. For example, the left subtree has prefixSum `{5: 1}`, if we have a `diff = 5` in right subtree, we should use this `hashmap[5]` from the left-subtree.

<br>

### Solution
Double DFS, Jun 22, 2026
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> int:
        # Q: Return # of ways in root to form targetSum
        # S: Run DFS and at each node, we run another dfs with this value, dfs(curSum + node.val), dfs(node.val)
        # If curSum == targetSum, update res
        # TC: O(n^2), n = total nodes, SC: O(H), height of the tree of recursion stack
        
        res = 0
        if not root:
            return 0
        def dfs(node, curSum):
            nonlocal res
            curSum += node.val
            if curSum == targetSum:
                res += 1
            # Exploring with previous node and start exploring from current node
            if node.left:
                dfs(node.left, curSum)
            if node.right:
                dfs(node.right, curSum)

        # Use this node to start dfs at each node freshly
        def dfs_helper(node):
            if not node:
                return
            dfs(node, 0)
            dfs_helper(node.left)
            dfs_helper(node.right)

        dfs_helper(root)
        return res
```
**Time Complexity:** $O(n^2)$, `n` is number of nodes in the tree. Since we are runnig DFS on each node, and runnnig DFS again to find targetSum. <br>
**Space Complexity:** $O(h)$

### Prefix
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> int:
        # Q: Return # of ways in root to form targetSum
        # S: Use prefixSum to calculate res += curSum - targerSum in hashmap
        # TC: O(n), n is total nodes, SC: O(n)

        hashmap = {0: 1}    # {curSum: cnt}
        res = 0
        def dfs(node, curSum):
            nonlocal res
            print(hashmap)
            if not node:
                return
            curSum += node.val
            # Update res
            if curSum - targetSum in hashmap:
                res += hashmap[curSum - targetSum]
            # Update hashmap
            hashmap[curSum] = hashmap.get(curSum, 0) + 1
            dfs(node.left, curSum)
            dfs(node.right, curSum)
            # Prevent to be used by another subtree
            hashmap[curSum] -= 1
        dfs(root, 0)
        return res
```
**Time Complexity:** $O(n)$, `n` is # nodes in the tree. <br>
**Space Complexity:** $O(n)$
