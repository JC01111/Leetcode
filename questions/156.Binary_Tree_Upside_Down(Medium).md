## 156. Binary Tree Upside Down (Medium)
**Date and Time:** Jun 2, 2026

Link: https://leetcode.com/problems/binary-tree-upside-down/

<br>

### Question:
Given the `root` of a binary tree, turn the tree upside down and return _the new root_.

You can turn a binary tree upside down with the following steps:

1. The original left child becomes the new root.
2. The original root becomes the new right child.
3. The original right child becomes the new left child.

The mentioned steps are done level by level. It is **guaranteed** that every right node has a sibling (a left node with the same parent) and has no children.

<br>

**Example 1:**

<img src="../images/156_1.jpg" width=500>

> **Input:** root = [1,2,3,4,5] <br>
> **Output:** [4,5,2,null,null,3,1]

**Example 2:**
> **Input:** root = [] <br>
> **Output:** []

**Example 3:**
> **Input:** root = [1] <br>
> **Output:** [1]

<br>

#### Constraints:
* The number of nodes in the tree will be in the range `[0, 10]`.

* `1 <= Node.val <= 10`

* Every right node in the tree has a sibling (a left node that shares the same parent).

* Every right node in the tree has no children.

<br>

### Walk-through: 
Recursively reverse the subtree **bottom up**.

1. Base case: if `root` is `None` or `root.left` is `None`, return `root` (no flipping needed).

2. Recurse down to the leftmost node — it becomes the `newRoot`.

3. On the way back up, rewire edges: `root.left.right = root` (old root becomes new right child) and `root.left.left = root.right` (old right sibling becomes new left child).

4. Sever the old edges by setting `root.left = root.right = None`.

<br>

### Python Solution:
```python
class Solution:
    def upsideDownBinaryTree(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
        # Q: Upside down trees
        # S: Recurisvely reverse subtree bottom up
        # 1. Use dummy node to save old node info
        # 2. Set newRoot = root.left, and set up new edges
        # 3. Remove old edges
        # TC:O(n), n is total nodes, SC: O(1)
        # Base case
        if not root or not root.left:
            return root
        newRoot = self.upsideDownBinaryTree(root.left)
        root.left.right = root
        root.left.left = root.right
        root.left = root.right = None
        return newRoot
```
**Time Complexity:** $O(n)$, `n` is the total number of nodes, each node is visited once. <br>
**Space Complexity:** $O(1)$, only constant extra space used (recursion stack is $O(h)$ where `h` is tree height, but at most $O(n)$).
