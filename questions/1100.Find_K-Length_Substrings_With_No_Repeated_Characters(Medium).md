## 1100. Find K-Length Substrings With No Repeated Characters (Medium)
**Date and Time:** May 26, 2026

Link: https://leetcode.com/problems/find-k-length-substrings-with-no-repeated-characters/

<br>

### Question:
Given a string `s` and an integer `k`, return _the number of substrings in `s` of length `k` with no repeated characters_.

<br>

**Example 1:**
> **Input:** s = "havefunonleetcode", k = 5 <br>
> **Output:** 6 <br>
> **Explanation:** There are 6 substrings they are: 'havef','avefu','vefun','efuno','etcod','tcode'.

**Example 2:**
> **Input:** s = "home", k = 5 <br>
> **Output:** 0 <br>
> **Explanation:** Notice k can be larger than the length of s. In this case, it is not possible to find any substring.

<br>

#### Constraints:
* `1 <= s.length <= 10^4`

* `s` consists of lowercase English letters.

* `1 <= k <= 10^4`

<br>

### Walk-through: 
Use a **sliding window** with two pointers `l, r` and a hashmap `dict` tracking character counts.

1. Expand the window by adding `s[r]` into `dict`.

2. If `dict[s[r]] > 1`, shrink from the left using a `while` loop until there are no repeated characters.

3. Once the window has exactly length `k` (i.e., `r - l + 1 == k`), increment `res` and shrink the window from the left by one to slide forward.

4. Increment `r` to expand the window.

<br>

### Python Solution:
May 26, 2026, [Time taken: 28m 6s]
```python
class Solution:
    def numKLenSubstrNoRepeats(self, s: str, k: int) -> int:
        # Use sliding window to add each char into dict {key: cnt}
        # When dict[k] > 1, use while loop to remove the first element until no repetition
        # TC: O(n), n = len(s), SC: O(n)
        res = 0
        l, r = 0, 0
        dict = {}
        while r < len(s):
            # Add each char into dict and check if it is repeated
            dict[s[r]] = dict.get(s[r], 0) + 1
            # Shrink the sliding window when repeated occur
            while dict[s[r]] > 1:
                dict[s[l]] = dict.get(s[l]) - 1
                l += 1
            # Check if the sliding window has length k, if so, res += 1
            if r - l + 1 == k:
                res += 1
                # Shrink the sliding window from the head
                dict[s[l]] = dict.get(s[l]) - 1
                l += 1
            # Expand sliding window
            r += 1
        return res
```
**Time Complexity:** $O(n)$, `n = len(s)`, each character is added and removed at most once. <br>
**Space Complexity:** $O(n)$, for the hashmap storing character counts.

### Python Solution:
May 31, 2026, [Time taken: 15m 42s]
```python
class Solution:
    def numKLenSubstrNoRepeats(self, s: str, k: int) -> int:
        # Q: Find # of substring with length k in s, no repeated chars
        # S: Use sliding window with hashmap to keep track of each char with its occurrence.
        # Expand r until r-l+1 == k, update res += 1
        # If repeated occurs, shrink l until no repetition
        # TC: O(n), n=len(s), SC: O(n)
        hashmap = {}
        l = 0
        res = 0
        for r in range(len(s)):
            # Add char into hashmap
            hashmap[s[r]] = hashmap.get(s[r], 0) + 1
            # Detect if repeated char occurs
            while l <= r and hashmap[s[r]] > 1:
                # Shrink sliding window (l ptr)
                hashmap[s[l]] -= 1
                l += 1
            # Check if length k substring exists
            if r - l + 1 == k:
                res += 1
                # Shrink sliding window
                hashmap[s[l]] -= 1
                l += 1
        return res
```
**Time Complexity:** $O(n)$, `n = len(s)`, each character is added and removed at most once. <br>
**Space Complexity:** $O(n)$, for the hashmap storing character counts.
