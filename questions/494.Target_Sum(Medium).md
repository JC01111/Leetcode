## 494. Target Sum (Medium)
**Date and Time:** Jun 14, 2026

Link: https://leetcode.com/problems/target-sum/

<br>

### Question:
You are given an integer array `nums` and an integer `target`.

You want to build an **expression** out of nums by adding one of the symbols `'+'` and `'-'` before each integer in nums and then concatenate all the integers.

* For example, if `nums = [2, 1]`, you can add a `'+'` before `2` and a `'-'` before `1` and concatenate them to build the expression `"+2-1"`.

Return _the number of different **expressions** that you can build, which evaluates to_ `target`.

<br>

**Example 1:**
> **Input:** nums = [1,1,1,1,1], target = 3 <br>
> **Output:** 5 <br>
> **Explanation:** There are 5 ways to assign symbols to make the sum of nums be target 3. <br>
> -1 + 1 + 1 + 1 + 1 = 3 <br>
> +1 - 1 + 1 + 1 + 1 = 3 <br>
> +1 + 1 - 1 + 1 + 1 = 3 <br>
> +1 + 1 + 1 - 1 + 1 = 3 <br>
> +1 + 1 + 1 + 1 - 1 = 3

**Example 2:**
> **Input:** nums = [1], target = 1 <br>
> **Output:** 1

<br>

#### Constraints:
* `1 <= nums.length <= 20`

* `0 <= nums[i] <= 1000`

* `0 <= sum(nums[i]) <= 1000`

* `-1000 <= target <= 1000`

<br>

### Walk-through:
Use **backtracking with memoization** (top-down DP). At each index `i`, try both `+nums[i]` and `-nums[i]`, accumulating a running `total`.

- **Base case:** when `i == len(nums)`, return `1` if `total == target`, else `0`.
- **Memoize** on `(total, i)`: the number of ways to reach `target` from index `i` with current running total.
- Both branches are summed: `backtrack(total + nums[i], i+1) + backtrack(total - nums[i], i+1)`.

The dp table has at most `n * (2 * totalSum + 1)` states where `totalSum = sum(nums)`.

<br>

### Python Solution:
```python
class Solution:
    def findTargetSumWays(self, nums: List[int], target: int) -> int:
        # Q: Return # of combinations we can make for nums to have sum == target
        # S: Backtrack for combination and check if the sum == target
        # TC: O(n*totalSum), n=len(nums), we create dp at each index, and each dp[i] we calculate the -totalSum and +totalSum, SC: O(n*totalSum)
        
        dp = {}     # {(total, i): value}
        def backtrack(total, i):
            # Base case
            if i == len(nums):
                return 1 if total == target else 0
            # Check memoization
            if (total, i) in dp:
                return dp[(total, i)]
            # How many ways to reach target at this position i
            results = backtrack(total + nums[i], i+1) + backtrack(total - nums[i], i+1)
            dp[(total, i)] = results
            return results
        return backtrack(0, 0)
```
**Time Complexity:** $O(n \cdot S)$, `n = len(nums)`, `S = sum(nums)`; at most `n * (2S+1)` unique `(index, total)` states. <br>
**Space Complexity:** $O(n \cdot S)$, for the memoization table and recursion call stack.
