## 527. Word Abbreviation (Hard)
**Date and Time:** Jun 12, 2026

Link: https://leetcode.com/problems/word-abbreviation/

<br>

### Question:
Given an array of distinct strings `words`, return _the **shortest abbreviation** of each word_. The following are the rules for forming an abbreviation:

* The **initial** abbreviation for each word is: first character + number of characters in between + last character. For example, `"internationalization"` → `"i18n"`.
* If more than one word shares the same abbreviation, each word's abbreviation is made **longer** by increasing the prefix length by 1. For example, `"internal"` → `"in6l"`, then `"int5l"`, and so on. Both words keep extending until their abbreviations are distinct.
* If the abbreviation is **not shorter** than the original word, keep the original word.

<br>

**Example 1:**
> **Input:** words = ["like","god","internal","me","internet","interval","intension","face","intrusion"] <br>
> **Output:** ["l2e","god","internal","me","i6t","interval","int4n","f2e","intr4n"]

**Example 2:**
> **Input:** words = ["aa","aaa"] <br>
> **Output:** ["aa","aaa"]

<br>

#### Constraints:
* `1 <= words.length <= 400`

* `2 <= words[i].length <= 400`

* `words[i]` consists of lowercase English letters.

* All strings in `words` are **unique**.

<br>

### Walk-through:
1. Helper `abbrev(word, i)` returns `word[:i] + str(len(word)-i-1) + word[-1]`, or the original word if the abbreviation isn't shorter (`i >= len(word) - 2`).

2. Initialize `res[i] = abbrev(words[i], 1)` and `prefix[i] = 1` for all words.

3. Repeat until no duplicates remain:
   - Build `hashmap {abbreviation: [indices]}`.
   - For any abbreviation with more than one index, increment `prefix[i]` for all conflicting words and recompute their abbreviations.

<br>

### Python Solution:
```python
class Solution:
    def wordsAbbreviation(self, words: List[str]) -> List[str]:
        # Q: Find abbreviation to shorten word
        # S: 1. Create a helper method for word abbreviation
        # 2. res[] to save all the word abbreviation
        # 3. prefixSum[] to save each word's index we currently used for current abbreviation
        # 4. Loop over res[] and create a hashmap for it {abbrevation: [indices of words]}
        # 5. Loop over hashmap to check if we have duplicate abbreviation of indices, if so, loop over all the indices for that abbreviation and update prefixSum for that index and update res[i] for new abbreviation.
        # Repeat 5. until no abbreviation in hashmap has more than 1 index
        # TC: O(n*k), n=len(words), k=avg length of word, SC: O(n*k)

        def abbrev(word, i):
            # Base case if i less than the word's second to the last index
            if i >= len(word)-2:
                return word
            # Slice the word
            return word[:i] + str(len(word) - i - 1) + word[-1]
        prefixSum = [0] * len(words)
        res = []
        # 1. First pass to fill out prefixSum and res
        for i, word in enumerate(words):
            prefixSum[i] += 1
            res.append(abbrev(word, 1))
        # 2. Iteratively loop over res and create hashmap to save {abbreviation: [indices]} until every abbr will only have 1 index
        while True:
            repeat = False
            hashmap = collections.defaultdict(list)     # {abbre: [indices]}
            # Fill hashmap
            for i, abbreviation in enumerate(res):
                hashmap[abbreviation].append(i)
            # Check hashmap
            for abbreviation, lst in hashmap.items():
                if len(lst) > 1:
                    repeat = True
                    # Update abbrev for each index in lst
                    for i in lst:
                        prefixSum[i] += 1
                        res[i] = abbrev(words[i], prefixSum[i])
            # If not repeat we can break
            if not repeat:
                break
        return res
```
**Time Complexity:** $O(n \cdot k)$, `n = len(words)`, `k` = average word length; each word's prefix grows at most `k` times before resolving all conflicts. <br>
**Space Complexity:** $O(n \cdot k)$, for storing all abbreviations in `res` and the hashmap.
