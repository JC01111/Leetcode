## 1233. Remove Sub-Folders from the Filesystem (Medium)
**Date and Time:** Jun 15, 2026

Link: https://leetcode.com/problems/remove-sub-folders-from-the-filesystem/

<br>

### Question:
Given a list of folders `folder`, return _the folders after removing all **sub-folders** in those folders_. You may return the answer in **any order**.

If a `folder[i]` is located within another `folder[j]`, it is called a **sub-folder** of it. A sub-folder of `folder[j]` must start with `folder[j]`, followed by a `"/"`.

For example, `"/a/b"` is a sub-folder of `"/a"`, but `"/b"` is not a sub-folder of `"/a/b/c"`.

The format of a path is one or more concatenated strings of the form: `'/'` followed by one or more lowercase English letters.

<br>

**Example 1:**
> **Input:** folder = ["/a","/a/b","/c/d","/c/d/e","/c/f"] <br>
> **Output:** ["/a","/c/d","/c/f"] <br>
> **Explanation:** Folders "/a/b" is a sub-folder of "/a" and "/c/d/e" is a sub-folder of "/c/d".

**Example 2:**
> **Input:** folder = ["/a","/a/b/c","/a/b/d"] <br>
> **Output:** ["/a"] <br>
> **Explanation:** Folders "/a/b/c" and "/a/b/d" will be removed because they are sub-folders of "/a".

**Example 3:**
> **Input:** folder = ["/a/b/c","/a/b/ca","/a/b/d"] <br>
> **Output:** ["/a/b/c","/a/b/ca","/a/b/d"]

<br>

#### Constraints:
* `1 <= folder.length <= 4 * 10^4`

* `2 <= folder[i].length <= 100`

* `folder[i]` contains only lowercase letters and `'/'`.

* `folder[i]` always starts with the character `'/'`.

* Each folder name is **unique**.

<br>

### Walk-through:
1. **Sort** `folder` lexicographically so that any parent folder always appears before its sub-folders.

2. For each path, split by `"/"` and incrementally rebuild the path segment by segment. At each step check whether the current prefix already exists in the `repeated` set.

3. If a prefix matches, the current path is a sub-folder — skip it. Otherwise add the full reconstructed path to `repeated` and the original path to `res`.

   Because all paths start with `"/"`, splitting produces `["", ...]` and the first rebuilt segment is always `"/"`, which is never stored in `repeated`. All meaningful prefix checks happen at deeper segments (e.g. `"//a"`, `"//a//b"`), which are consistent across all paths.

<br>

### Python Solution:
```python
class Solution:
    def removeSubfolders(self, folder: List[str]) -> List[str]:
        # S: 1. Strip word by delimiter "/"
        # 2. Save chars as key in hashmap
        # TC:O (nlogn + n*k), n=len(folder), k is avg length of folder[i], SC: O(n*k)
        
        folder.sort()
        repeated = set()
        res = []
        for path in folder:
            tmp = ""
            newPath = path.split("/")
            repeat = False
            for c in newPath:
                tmp = tmp + "/" + c
                if tmp in repeated:
                    repeat = True
                    break
            if not repeat:
                repeated.add(tmp)
                res.append(path)
        return res
```
**Time Complexity:** $O(n \log n + n \cdot k)$, `n = len(folder)`, `k` = average path length; sorting is $O(n \log n)$ and processing each path segment-by-segment is $O(n \cdot k)$. <br>
**Space Complexity:** $O(n \cdot k)$, for the `repeated` set storing all non-sub-folder paths.

<br>

### Trie Solution:
Build a Trie from all folder paths (split by `"/"`). In a second pass, walk each path through the Trie — if any intermediate node (not the last segment) has `end = True`, the current path is a sub-folder and is skipped.

```python
class Node:
    def __init__(self):
        self.children = {}
        self.end = False

class Solution:
    def removeSubfolders(self, folder: List[str]) -> List[str]:
        # S: 1. Create Trie by checking all folder
        # 2. For a path in folder, if we have self.end = True in path, we should break and not add this path into res[]
        # TC: O(n*l), n=len(folder), l = max(len(path)), SC: O(n*l)
        
        # 1. Process folder into trie
        trie = Node()
        for path in folder:
            path = path.split("/")
            # Build trie
            curr = trie
            for c in path:
                if c not in curr.children:
                    curr.children[c] = Node()
                curr = curr.children[c]
            # Set the last char's end = True
            curr.end = True

        # 2. Another pass to check sub-folders and update res[]
        res = []
        for path in folder:
            processed = path.split("/")
            curr = trie
            repeat = False
            for c in processed:
                # If char is not the last char but .end = True
                if c in curr.children and c != processed[-1] and curr.children[c].end:
                    repeat = True
                    break
                curr = curr.children[c]
            if not repeat: 
                res.append(path)
        return res
```
**Time Complexity:** $O(n \cdot l)$, `n = len(folder)`, `l` = max path length; building and querying the Trie each take $O(n \cdot l)$. <br>
**Space Complexity:** $O(n \cdot l)$, for the Trie nodes.
