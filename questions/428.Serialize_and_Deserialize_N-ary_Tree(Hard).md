## 428. Serialize and Deserialize N-ary Tree (Hard)
**Date and Time:** Jun 23, 2026

Link: https://leetcode.com/problems/serialize-and-deserialize-n-ary-tree/

<br>

### Question:
Serialization is the process of converting a data structure or object into a sequence of bits so that it can be stored in a file or memory buffer, or transmitted across a network connection link to be reconstructed later in the same or another computer environment.

Design an algorithm to serialize and deserialize an **N-ary tree**. An N-ary tree is a rooted tree in which each node has no more than N children. There is no restriction on how your serialization/deserialization algorithm should work. You just need to ensure that an N-ary tree can be serialized to a string and this string can be deserialized to the original tree structure.

<br>

**Example 1:**
> **Input:** root = [1,null,3,2,4,null,5,6]
> 
> **Output:** [1,null,3,2,4,null,5,6]

**Example 2:**
> **Input:** root = [1,null,2,3,4,5,null,null,6,7,null,8,null,9,10,null,null,11,null,12,null,13,null,null,14]
> 
> **Output:** [1,null,2,3,4,5,null,null,6,7,null,8,null,9,10,null,null,11,null,12,null,13,null,null,14]

<br>

#### Constraints:
* The number of nodes in the tree is in the range `[0, 10^4]`.

* `0 <= Node.val <= 10^4`

* The height of the n-ary tree is less than or equal to `1000`.

* Do not use class member/global/static variables to store states. Your encode and decode algorithms should be stateless.

<br>

### Walk-through:
**`serialize`:** Run pre-order DFS. For each node, append its `val` and `len(children)` to `res`. The child count tells `deserialize` exactly how many recursive calls to make for that node — no delimiter needed.

**`deserialize`:** Use a global index `i` into the split token list. At each call, read `data[i]` as the node value and `data[i+1]` as the number of children, advance `i += 2`, then loop `numChildren` times calling `dfs()` recursively to build each child.

<br>

### My Solution:
```python
class Codec:
    def serialize(self, root: 'Node') -> str:
        # Q: Convert root to a string
        # S: Pre-order to save each node with its number of children
        res = []
        def dfs(node):
            nonlocal res
            if not node:
                return
            res.append(str(node.val))
            res.append(str(len(node.children)))
            # Run DFS on node children
            for child in node.children:
                dfs(child)
        dfs(root)
        return " ".join(res)
	
    def deserialize(self, data: str) -> 'Node':
        # S: Run DFS on each node's children
        if not data:
            return None
        i = 0       # global var to access children
        data = data.split()
        def dfs():
            nonlocal i
            # Get new node from global i
            node = Node(int(data[i]))
            numChildren = int(data[i+1])
            i += 2      # Get the next node either for current node's children or build next node
            # Build children by repeating
            for _ in range(numChildren):
                node.children.append(dfs())
            return node
        return dfs()

```
**Time Complexity:** $O(n)$, where `n` is the number of nodes <br>
**Space Complexity:** $O(n)$
