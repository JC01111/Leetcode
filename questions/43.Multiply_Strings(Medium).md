## 43. Multiply Strings (Medium)
**Date and Time:** Jun 7, 2026

Link: https://leetcode.com/problems/multiply-strings/

<br>

### Question:
Given two non-negative integers `num1` and `num2` represented as strings, return the product of `num1` and `num2`, also represented as a string.

**Note:** You must not use any built-in BigInteger library or convert the inputs to integer directly.

<br>

**Example 1:**
> **Input:** num1 = "2", num2 = "3" <br>
> **Output:** "6"

**Example 2:**
> **Input:** num1 = "123", num2 = "456" <br>
> **Output:** "56088"

<br>

#### Constraints:
* `1 <= num1.length, num2.length <= 200`

* `num1` and `num2` consist of digits only.

* Both `num1` and `num2` do not contain any leading zero, except the number `0` itself.

<br>

### Walk-through: 
Use a result array of length `len(num1) + len(num2)` and simulate digit-by-digit multiplication.

For digits at positions `i` (from `num2`) and `j` (from `num1`), their product lands at index `i + j + 1`, with carry propagating to `i + j`.

1. Handle edge case: if either input is `'0'`, return `'0'`.

2. Iterate `i` from the end of `num2` and `j` from the end of `num1`. Compute `digit = num1[j] * num2[i]`, add to `lst[i+j+1]`, propagate carry to `lst[i+j]`, and mod `lst[i+j+1]` by 10.

3. Strip the leading zero at `lst[0]` if present, then join into a string.

```
     1 2 3    <- num1
       5 6    <- num2
 ---------
     7 3 8    <- i=1 (digit 6)
   6 1 5      <- i=0 (digit 5)
 ---------
 0 6 8 8 8
```

<br>

### Python Solution:
```python
class Solution:
    def multiply(self, num1: str, num2: str) -> str:
        # Q: Return product of num1 and num2 without using large multiplication (only digit to digit)
        # S: Use a list to initialize the value at each index with carry, list with at most length len(num1) + len(num2)
        # TC: O(m*n), m=len(num1), n=len(num2), SC: O(m+n)
        if num1 == '0' or num2 == '0':
            return '0'
        lst = [0] * (len(num1) + len(num2))
        for i in range(len(num2)-1, -1, -1):
            for j in range(len(num1)-1, -1, -1):
                digit = int(num1[j]) * int(num2[i])
                # i + j + 1 to access digit
                lst[i+j+1] += digit
                # If > 10, get carry
                lst[i+j] += lst[i+j+1] // 10
                # If > 10, remove 10
                lst[i+j+1] %= 10
        # Check if lst[0] == 0
        if lst[0] == 0:
            lst = lst[1:]
        return "".join(str(digit) for digit in lst)
```
**Time Complexity:** $O(m \cdot n)$, `m = len(num1)`, `n = len(num2)`, nested loop over all digit pairs. <br>
**Space Complexity:** $O(m + n)$, for the result array.
